# cloc-c：C语言代码行数统计工具

## 项目简介
cloc-c 是一个使用 C 语言编写的命令行代码行数统计工具，支持单文件统计、递归目录遍历、多行注释识别和汇总输出。

## 功能特性
- 统计单个文件的总行数、空行数、注释行数
- 支持 `//` 单行注释和 `/* */` 多行注释
- 忽略字符串内部的 `//` 误判（如 `"http://"`）
- 递归遍历目录，统计所有 `.c` / `.h` 文件
- 输出每个文件的明细统计 + 汇总总计（总文件数、总行数、总空行数、总注释行数）
- 跨平台支持：Windows（MSVC/MinGW）和 Linux（gcc）/ Alpine（Docker）

## 编译

### 方式一：使用 Makefile（推荐）
```bash
make build
```

### 方式二：直接使用 gcc
```bash
gcc -Wall -Wextra -o cloc main.c
```

## 使用

```bash
# 统计当前目录下所有 .c / .h 文件
./cloc .

# 统计指定目录
./cloc -path ./src

# 统计单个文件（兼容模式）
./cloc main.c
```

## 输出示例
```
./src/main.c: 总行数: 10, 空行数: 2, 注释行数: 3
./src/utils.h: 总行数: 5, 空行数: 0, 注释行数: 1
./src/sub/helper.c: 总行数: 8, 空行数: 1, 注释行数: 2

汇总: 文件数: 3, 总行数: 23, 空行数: 3, 注释行数: 6
```

## Docker 运行

```bash
# 构建镜像
docker build -t cloc-c .

# 运行容器（统计 /app 目录）
docker run cloc-c
```
